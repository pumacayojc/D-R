# TRL4 – Transporte D&R (Binder / JupyterLab Web)

Este repositorio está listo para ejecutarse en **Binder** (JupyterLab en la web) o en JupyterLab local.

## Ejecutar en Binder (JupyterLab Web)
1. Sube el contenido de este repositorio a GitHub.
2. Abre Binder con esta URL (reemplaza `USUARIO/REPO` por tu GitHub):

```text
https://mybinder.org/v2/gh/USUARIO/REPO/HEAD?urlpath=lab
```

3. En JupyterLab (Binder), abre:
- `notebooks/TRL4_Transporte_DR.ipynb`

4. Ejecuta: **Run → Run All Cells**.

## Estructura
- `data/` → CSV de entrada
- `notebooks/` → notebook principal
- `requirements.txt` → dependencias para Binder

## Nota
Binder es temporal: los cambios dentro de Binder no se guardan automáticamente en GitHub.
